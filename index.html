Hipervinculos

Los hipervinculos nos ayudan para poder transefirnos de una pagina a otra o tambien de una parte del sitio a otra la etiqueta se llama < a href="(link)"> < /a > asi como el link que presionaron en la pagina anterior para direccionarlos aqui.

Ejemplo de la etiqueta: < a href="https://www.google.com/" >Google< /a >

Google

Imagenes y Marquesinas

La etiqueta para poner imagenes es la siguiente <"img" src="(imagen descargada y puesta en la carpeta)"> <"/img">

Ejemplo de la etiqueta: < img src="a.jpeg" >

Con las marquesinas practicamente es lo mismo solo que ahora puedes hacer que se mueva la imagen o un texto

Ejemplo de texto en movimiento: < marquee >< p >PROGRAMACIÓN< /p >< /marquee >

PROGRAMACIÓN


E igual con las imagenes puedeshacer que se muevan.

Ejemplo de imagen con marquesina de movimiento: < marquee direction="right" >< img src="a.jpeg" >< /marquee >

Listas

Para el comando de listas podemos variar entre las etiquetas <"ol"> <"/ol"> (listas ordenadas o numeradas) y <"ul"> <"/ul"> (listas sin un orden) y la etiqueta <"li"> para poner un elemento en la lista

Ejemplo de la etiqueta <"ol">

  1. Aries
  2. Tauro
  3. Géminis
  4. Cáncer
  5. Leo
  6. Virgo
  7. Libra
  8. Escorpión
  9. Sagitario
  10. Capricornio
  11. Acuario
  12. Piscis

Ejemplo de la etiqueta <"ul">

  • Dos

  • Diez

Tablas

Con la etiqueta < table > puedes crear tablas y utilizar las etiquetas < tr > y < td > para las columnas y las filas

Nota!!!: Despues del titulo si quieres cambiar el color de las tablas puedes hacer lo siguiente:

Ejemplo de las etiquetas utilizadas en las tablas:

< style > table, tr, td { border: 1px solid lightseagreen; } < /style >
